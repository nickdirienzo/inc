# Clear Attention

Clear the needs_attention flag from an epic after you've handled the request.

## Usage

```bash
inc attention clear <epic-id>
```

## When to Use

- After answering an agent's question
- After approving a spec or plan
- After providing feedback that resolves the request

## Example

```bash
inc attention clear 1ec512f4
```
